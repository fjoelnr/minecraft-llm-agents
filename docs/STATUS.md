# Status

## Project State

`minecraft-llm-agents` is an active prototype repository.

The core stack is already visible:

- FastAPI orchestrator
- MCP-shaped context
- vector memory support
- A2A-oriented coordination primitives
- Node.js gateway entry point

## What Works

- local orchestrator startup
- API and schema surface
- Python test suite
- gateway development entry point
- starter documentation for setup and experiments

## What Still Needs Work

- stronger onboarding for non-authors
- clearer separation between implemented features and planned features
- screenshots or walkthrough examples for experiments
- more end-to-end integration validation

## Current Risk Areas

- the gateway is still intentionally thin
- Mineflayer-facing integration is not yet the main strength of the repo
- the docs are better than before, but still builder-oriented
